Usage of this API

  • GET Users

    /users

    returns json object with all users, their commands and data

  • GET Command

    /get/:user

    returns the command for a specific user

  • POST Data

    /data/:user

    post arbitrary data as a specific user

  • POST Command

    /command

    post a new object for a specific user

  • POST File

    /file

    post a file, that is saved on the server

  • GET File

    /download

    download the file, that was saved on the server with POST FILE
